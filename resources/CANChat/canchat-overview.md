# CANChat

CANChat is SSC's GC-approved generative AI chatbot for Government of Canada employees.
It is designed for everyday, **unclassified** productivity tasks and is hosted within
the SSC environment to align with Canadian data sovereignty and GC policy requirements.

Think of it as a GC-approved "ChatGPT-like assistant" for internal work.

## What it is

- An LLM-based chatbot built and operated by Shared Services Canada
- Available to GC employees for unclassified knowledge work
- Hosted within the GC/SSC environment — not a public cloud consumer service
- Designed to align with GC policies, standards, and data residency requirements

## What you can use it for

| Task | Examples |
|------|---------|
| Drafting | Emails, memos, reports, architecture writeups |
| Rewriting | Improve tone, clarity, brevity of existing text |
| Summarizing | Long documents, meeting notes, technical specs |
| Research | Quick background on topics, brainstorming options |
| Meeting prep | Agendas, action item summaries |
| Office tasks | Excel formula help, ConOps drafts, RACI tables |

## How to access

1. Go to the CANChat web application (SSC internal access).
2. Accept the Terms of Use on first login.
3. Optionally install the desktop app if available for your environment.

## Key features

- Chat interface similar to ChatGPT — enter a prompt, receive generated text
- Document upload for analysis
- Saved/reusable prompts
- Session management
- In-app help and training resources

## Responsible use rules

**You must follow these rules when using CANChat:**

| Rule | Detail |
|------|--------|
| Unclassified only | Only use for Protected A or below / unclassified work |
| No sensitive data | Do not input personal, protected, or classified information |
| Validate outputs | Always review and verify AI-generated content before using it |
| No decision automation | Do not use outputs to make consequential decisions without human review |

## How it differs from public AI tools

| Aspect | CANChat | Public tools (ChatGPT, etc.) |
|--------|---------|------------------------------|
| Hosting | SSC / GC environment | Commercial cloud |
| Data sovereignty | Canadian-controlled | Vendor-controlled |
| Policy alignment | GC-specific | General purpose |
| Access | GC employees only | Public |
